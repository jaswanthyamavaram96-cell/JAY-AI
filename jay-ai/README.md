# Jay AI – Static Website

A modern, professional, and futuristic landing page for Jay AI: an all‑in‑one super AI assistant for search, images, video, music, voice, and coding.

## Features
- Hero with animated particle background and gradient glow
- Interactive features grid with hover effects
- Showcase: Q&A demo, image/video mocks, music sample
- Engineer’s Day special section with looping tribute video and quiz
- "Why Jay AI?" comparison, testimonials, and CTA
- Smooth scroll, reveal‑on‑scroll animations
- Dark/Light theme toggle with preference saved
- Fully responsive (mobile, tablet, desktop)

## Getting Started
Open `index.html` directly in a browser, or serve locally for best results.

### Simple local server (Python)
```bash
cd /workspace/jay-ai
python3 -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

## Customization
- Text/content: edit `index.html`
- Colors/gradients/spacing: edit CSS variables in `styles.css`
- Animations and interactions: edit `script.js`
- Favicon: replace `assets/favicon.svg`

## Deploy
- Static hosting options: GitHub Pages, Netlify, Vercel, Cloudflare Pages
- Deploy the root directory containing `index.html`, `styles.css`, and `script.js`
